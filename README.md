# Missão Matemática — 6º Ano

Jogo educativo de matemática criado para alunos do 6º ano do Ensino Fundamental.

## Conteúdos

- Operações fundamentais
- Frações
- Números decimais
- Porcentagem
- Equações simples
- Área e perímetro

## Como jogar

Abra o arquivo `index.html` ou publique o repositório com o GitHub Pages.

## Inserção no Genially

Depois de ativar o GitHub Pages, use o endereço público do jogo no recurso **Inserir conteúdo externo** do Genially.

Desenvolvido para funcionar em celulares e computadores.
